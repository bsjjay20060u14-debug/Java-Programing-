# Java-Programing-Assignment
대학교에서 자바 스크립트 수업을 이수 중인 2학년 학생들을 대상으로 학기 말에 진행된 과제입니다.


해당 과제는 사전에 미리 구현되어야하는 기능들을 요구받고 그 사항을 직접 코드로 구현해내는 것을 목표로하는 과제였습니다.


과제에서 요구한 것은 파일 입출력 모소드의 컬렉션을 사용하여 파일 내용을 읽고 쓰고 관리하는 프로그램을 만드는 것이였습니다.


총 7가지의 기능이 요구되었고 설명은 아래와 같습니다.

> ### 실행
프로그램을 실행하면 1~7번의 메뉴가 나열되어야합니다.


메뉴의 목록을 보고 원하는 메뉴의 넘버를 입력하면 선택된 메뉴가 실행됩니다.


메뉴 목록은 아래와 같습니다.


`1. 자기 소개`


`2. 학생성적 입력`


`3. 학생성적 학점 출력`


`4. 학생 검색`


`5. 계산기 프레임`


`6. 학생 주소록 저장, 출력`


`7. 끝내기`


> ### 메뉴 1

"자기 소개"라는 이름의 메뉴입니다.


사용자가 '1' 을 입력하면, 준비되어있는 자기 소개가 출력되는 기능을 가진 메뉴 입니다.

> ### 메뉴 2

"학생 성적 입력"이라는 이름의 메뉴입니다.


사용자가 '2' 를 입력하면, 학생의 이름과 그 학생의 중간, 기말, 과제의 성적을 입력하라는 메시지가 출력됩니다.


그렇게 총 5명의 정보를 입력하면 입력이 완료되었다는 메시지가 출력됩니다. 입력된 정보는 c\temp\score.txt에 그대로 저장됩니다.

> ### 메뉴 3

"학생성적 학점 출력"이라는 이름의 메뉴입니다.


사용자가 '3' 을 입력하면, 2번에 의해 저장되어있던 정보가 그대로 출력됩니다. 만약 2번 메뉴의 절차를 생략해 학생 정보의 파일이 없다면 입력된 정보가 없다는 메시지를 출력합니다.


입력받은 점수들을 모두 합산해 전체 합을 기준으로 각각 5점씩 끊어 A+ 부터 F 까지 점수를 자동으로 출력시키도록 구현했습니다.

> ### 메뉴 4

"학생 검색"이라는 이름의 메뉴입니다.


사용자가 '4' 를 입력하면, 검색을 원하는 학생의 이름을 요구하는 메시지가 출력됩니다. 학생의 이름을 입력하면 2번 메뉴에서 저장된 정보와 비교해 일치하는 이름이 있을 경우 학생의 정보를 출력하고, 그렇지 않은 경우 존재하지 않는 학생이라는 메시지를 출력합니다.


"그만" 을 입력하면 종료됩니다.

> ### 메뉴 5

"계산기 프레임"이라는 이름의 메뉴입니다.


사용자가 '5' 를 입력하면, 간단한 연산 기능을 지원하는 계산기가 실행됩니다. 


CE버튼을 누르면 이전에 입력된 값들이 초기화되고 계산버튼을 누르면 입력한 식의 답이 결과란에 출력됩니다.


해당 계산기는 간단한 사칙연산은 가능하지만 2개 이상의 연산자가 요구되는 복잡한 식을 처리할 순 없다는 단점이 있습니다.
